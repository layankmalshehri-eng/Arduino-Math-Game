# 🎮 Arduino Math Game

لعبة رياضيات تفاعلية باستخدام Arduino، شاشة LCD، كيباد، وبازر 🔊

## ✨ المميزات

* 🎯 3 مستويات (سهل - متوسط - صعب)
* ➕➖✖➗ عمليات حسابية متنوعة
* ⏱ مؤقت 40 ثانية لكل سؤال
* ⭐ نظام نقاط
* 🔊 مؤثرات صوتية (فوز / خسارة)
* ⌨️ إدخال باستخدام Keypad
* ❌ زر حذف (*)
* ✅ زر تأكيد (#)

---

## 🧰 المكونات

* Arduino Uno
* LCD 16x2 (I2C)
* Keypad 4x4
* Buzzer

---

## 🔌 التوصيل

### شاشة LCD (I2C)

| LCD | Arduino |
| --- | ------- |
| GND | GND     |
| VCC | 5V      |
| SDA | A4      |
| SCL | A5      |

---

### Keypad

| Keypad | Arduino |
| ------ | ------- |
| R1     | 9       |
| R2     | 8       |
| R3     | 7       |
| R4     | 6       |
| C1     | 5       |
| C2     | 4       |
| C3     | 3       |
| C4     | 2       |

---

### Buzzer

| Buzzer | Arduino |
| ------ | ------- |
| +      | 10      |
| -      | GND     |

---

## 🚀 طريقة التشغيل

1. تحميل الكود إلى Arduino
2. توصيل المكونات
3. تشغيل الجهاز
4. اضغط A لبدء اللعبة 🎮
